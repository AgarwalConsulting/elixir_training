# Operators Cheatsheet

## Strings

- `<>` concatenation

- `"#{<val>}"` interpolation

## Arithmetic

- `+`
- `-`
- `*`
- `/`
- `div`
- `rem`

## Boolean

- `&&` vs `and`
- `||` vs `or`
- `!` vs `not`

## Comparison

- `==` vs `===`

## Lists

- `++`
- `--`
- `hd`, `tl`

## Pattern Matching

- `=` is matching operator

- `|`

  Eg. [head | tail] = [1, 2, 3]

  - Can also be used for prepending

- `^` is the pin operator

  - doesn't rebound the value, just matches

- `_` to ignore
